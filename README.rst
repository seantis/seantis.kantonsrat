Seantis Kantonsrat
==================

Specialized type and custom organisations for the Kantonsrat of Zug. If you
have never heard of a Kantonsrat, this is in no way meant for you.

Requirements
------------

-  Python 2.7
-  Plone 4.3+
-  Linux / Posix ( Windows may or may not work )
